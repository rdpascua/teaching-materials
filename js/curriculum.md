# JavaScript Quest: 30-Day Bootcamp

### From Village Noob to Corporate Hero

**Format:** Max 30 minutes per day | **Prerequisites:** None — starts from zero

> You start as a young coder in a small pixel village. Each day you learn a new
> spell (concept), fight a monster (exercise), and earn XP. By Day 22 you get
> "hired" and the quests become real-world corporate missions.

---

## The Story Arc

```
Days 01–07   THE VILLAGE        Learn the basics, earn your first blade
Days 08–14   THE FOREST         Harder creatures, sharper tools
Days 15–21   THE DUNGEON        Face the browser, conquer async dragons
Days 22–28   THE CORPORATION    You got hired — ship real features
Days 29–30   THE FINAL BOSS     Capstone project & career launch
```

|-----|-------------|-----------------|----------|

## XP & Progression System

Each day students earn XP for participation:

| Activity | XP |
|----------|----|
| Attend the lesson | +10 XP |
| Complete the exercise | +20 XP |
| Solve the bonus challenge | +15 XP |
| Beat the Boss Fight | +50 XP |

```
Ranks:
  0–50   XP  →  Villager
 51–150  XP  →  Apprentice
151–300  XP  →  Adventurer
301–500  XP  →  Knight
501–700  XP  →  Wizard
701–900  XP  →  Hero
900+     XP  →  Legend
```

Max possible XP: **30×(10+20+15) + 4×50 = 1,350 + 200 = 1,550 XP**

## Lesson Structure (30 min max)

```
~5  min   THE STORY    — Set the scene, introduce the quest
~10 min   THE SPELL    — Teach the concept with analogies & code
~10 min   THE BATTLE   — Live coding demo / walkthrough
~5  min   THE LOOT     — Exercise handout + bonus challenge
```

## File Structure Per Day

```
day-XX/
  notes.md        — Lesson script with story + talking points
  examples.js     — Runnable code demos
  exercises.js    — Practice problems (the "battles")
  solutions.js    — Solutions (revealed after)
```
