## JavaScript Quest: Instructor Guide

A **30-day gamified bootcamp** (max 30 min/day). Students progress from RPG village noob to corporate developer.

### Story Arc

| Days | Arc | Tone |
|------|-----|------|
| 00 | **Prologue** | 10-min briefer — JS history, setup |
| 01–07 | **The Village** | Fun RPG — basics (variables, types, loops, functions) |
| 08–14 | **The Forest** | Harder quests — arrays, objects, closures, boss fight |
| 15–21 | **The Dungeon** | Browser combat — DOM, events, promises, async/await |
| 22–28 | **The Corporation** | Hired at NovaTech Inc. — Jira tickets, real-world JS |
| 29–30 | **The Final Boss** | Capstone project + graduation |

### Files Per Day

```
day-XX/
  slides.md       Presentation for NotebookLM (upload this)
  discussion.md   Full lesson script with talking points
  examples.js     Runnable code demos
  exercises.js    5 problems + 1 bonus challenge
  solutions.js    Complete solutions
```

### How to Use

1. Upload `slides.md` to **NotebookLM** for presentation mode
2. Use `discussion.md` as your teaching script (timed sections included)
3. Run `examples.js` live during the demo portion
4. Hand out `exercises.js` — students work through problems
5. Reveal `solutions.js` after the exercise

### XP System

Students earn XP per day: +10 attend, +20 exercise, +15 bonus, +50 boss fights. Track ranks from Villager (0 XP) to Legend (900+ XP).
